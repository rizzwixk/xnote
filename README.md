# xnote

A minimal, grayscale notes app built with Electron.

## Usage

```bash
npm install
npm run dev
```

## Keyboard Shortcuts

| Shortcut | Action       |
|----------|--------------|
| Ctrl+N   | New note     |
| Ctrl+D   | Delete note  |

## NOTE

this app is highly experimental and just a base for a future notes app that has a bunch of ai features
